# AI & Water Usage Analysis

## Project Goal

Investigate the relationship between AI growth, data center expansion, and water stress across regions.

## Research Questions

1. How much water do AI workloads consume?
2. Where are major AI data centers located?
3. Are AI facilities concentrated in water-stressed regions?
4. What sustainability challenges may arise from AI expansion?

## Tools

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Project Structure

data/ - Raw and processed datasets

notebooks/ - Exploratory analysis

src/ - Reusable Python code

outputs/ - Visualizations and reports

## Project Roadmap

### Phase 1
- Collect water stress datasets
- Explore data quality
- Perform cleaning

### Phase 2
- Analyze regional water stress
- Identify high-risk regions

### Phase 3
- Gather AI and data center information
- Compare infrastructure growth with water stress

### Phase 4
- Visualize findings
- Create final report and dashboard

### Phase 5
- Publish results on GitHub and LinkedIn
